### Featured projects
 
**[zotero_MCP](https://github.com/restjohnson/zotero_MCP)** &nbsp;`Python`
 
A custom Model Context Protocol (MCP) server that connects a Zotero research library to Claude Desktop. Supports full read and write access: search by keyword, browse collections, retrieve citation metadata, tag papers, and pull PDFs directly into conversation. Built for researchers who want an AI-native literature workflow.
 
**[ns_railroad_time_series](https://github.com/restjohnson/ns_railroad_time_series)** &nbsp;`Python · Jupyter`
 
Time series analysis on Norfolk Southern railroad data. Explores forecasting and trend detection using classical and ML-based approaches to surface operational patterns in large-scale transportation datasets.
 
**[financial-credit-prediction](https://github.com/restjohnson/financial-credit-prediction)** &nbsp;`Python · Jupyter`
 
A machine learning classification pipeline for financial credit risk prediction. Covers exploratory data analysis, feature engineering, model selection, and evaluation across metrics like AUC-ROC, precision, and recall.
 
**[phdtracker](https://github.com/restjohnson/phdtracker)** &nbsp;`JavaScript · MERN`
 
A full-stack PhD project management system built on MongoDB, Express, React, and Node. Tracks milestones, tasks, and team roles across a research timeline.
